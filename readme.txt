Все Java-проекты выполнены в IDE Eclipse Photon June 2018 (информация на случай проблем с кодировкой (CP1251): некоторые комментарии на русском языке).

В папке task_1_3 лежат исходники заданий 1 и 3. Запускать из класса Main соответствующего пакета.
В задании 3 используется драйвер jdbc-3.7.2 для общения с базой данных SQLite.

В папке task_2 лежит файл с базой данных SQLite, в ней сохранены запросы и таблицы. Также запросы есть в файле sqlqueries.txt.
Также можно найти папку sqliteadmin, этой программой можно открыть базы данных.
