Для просмотра и редактирования базы данных не из Java использовалась утилита sqliteadmin, её можно найти в соответствующей папке.
Внутри базы данных "bugtracker_db.s3db" сохранены таблицы и запросы на создание таблиц, добавление информации и выборку всех багов, с указанием проекта и пользователя.

Также в текущей папке есть файл "sql_queries.txt", в нём сохранены те же запросы, которые есть в базе данных.
Кроме того, графическая схема базы данных есть на скриншоте, лежащем в текущей папке.
